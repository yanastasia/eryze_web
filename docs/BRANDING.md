# Eryze Brand System

Keywords: Rise • Craft • Motion • Clarity • Impact

## Voice
- Essence: forward, luminous, creative.
- Tone: professional × human; poetic without fluff.
- Headlines: actions or beliefs, never plain descriptions.

## Typography
- Body: Plus Jakarta Sans (Google) — modern, clean, friendly.
  - CSS variable: `--font-body`
- Header: Space Grotesk — confident geometry for titles.
  - CSS variable: `--font-header`
- Mono: Geist Mono — technical accents.
  - CSS variable: `--font-geist-mono`

## Color
- Background (dark): `#0B1224`
- Accent (purple): `#8B5CF6` with lighter `#C084FC`
- Accent (cyan): `#79CFFF`
- Foreground: Tailwind `foreground` and `muted-foreground` tokens.

## Motion Principles
- Purposeful, not decorative.
- Fluid hand-offs between sections.
- Honor `prefers-reduced-motion`.
- Parallax depth on hero; subtle particle ambience.

## Components
- Hero, About, Services, Projects, Labs, Founder, CTA, Footer.
- AmbientParticles supports `circle` and `hex` shapes (hex used primarily in Labs).

## Assets
- Logo light/dark, gradient backgrounds, founder portrait, icon set.
- Favicon: `public/favicon.ico` (placeholder; to be updated).

## Copy Examples
- "Rise above the ordinary. Build what moves the world."
- "Craft with impact" (Work)
- "Ideas in motion" (Labs)
- "Collaborate with clarity" (Contact)
